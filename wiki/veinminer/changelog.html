Changelog

This page lists all notable changes to the VeinMiner Add-On

  • Initial release of VeinMiner
  • Vein mining functionality for all vanilla ores
  • Fortune and Silk Touch enchantment support
  • Smart vein detection with visual outline system
  • Durability protection system
  • Support for vanilla pickaxes
  • Add-On compatibility for custom ores
  • Configurable settings for activation method and tool usage
  • Supported Add-Ons:
    • Tinker's Construct
    • More Ores and Tools
    • Weapons+
