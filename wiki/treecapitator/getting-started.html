Getting Started with TreeCapitator

Welcome to TreeCapitator! This guide will help you quickly set up and start using this powerful tree-felling add-on for Minecraft Bedrock Edition.


1. Installation
  1. Purchase the TreeCapitator Add-On from the Minecraft Marketplace
  2. Click on "Activate Add-On", then either select an existing world or create a new one.
  3. That's all there is to do!
2. Crafting the TreeCapitator Book

To access TreeCapitator's settings in-game, you'll need to craft the configuration book:

  1. Open your crafting table.
  2. Place a regular book in the crafting grid.
  3. Place any sapling next to the book.
  4. Collect the TreeCapitator configuration book from the result slot.
Note: You already receive a TreeCapitator book by default, so you only have to craft one if you ever lose it!
3. Basic Usage
  1. Equip any axe in your main hand.
  2. Approach a tree.
  3. Break the bottom-most log of the tree.
  4. Watch as the entire tree falls!
Note: By default, TreeCapitator activates when you're standing. You can change this in the settings to activate while sneaking instead.
4. Configuring TreeCapitator
  1. Hold the TreeCapitator configuration book in your main hand.
  2. Use (right-click) the book to open the configuration menu.
  3. Navigate through the settings to customize TreeCapitator to your liking.
5. Compatibility

TreeCapitator is designed to work with vanilla Minecraft trees and many custom trees from other Add-Ons.
Check the compatibility page for more details.

Congratulations! You're now ready to start using TreeCapitator. Happy tree felling!
